Galaxy.getEventMapOf

returns a detail data of a module's events

Usage

Galaxy.getEventMapOf(moduleName)

Parameter Type Description
moduleName String name of the mdoule

Return

Parameter Type Description
publishedEvents Undefined or Object a list of published events
queriedRequest Undefined or Object a list of queried requests
receivedEvents Undefined or Object a list of received events
reservedEvents Undefined or Object a list of reserved events
respondedResponse Undefined or Object a list of responded responses
subscribedEvents Undefined or Object a list of subscribed events
